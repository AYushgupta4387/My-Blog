SHORT CIRCUITING (&& AND ||)

|| operator -

It is the same as logical OR operator. It will check the first value, if that's true, it will return that value, if it's false, it will move to the next value. It will continue doing this until a truthy value is reached. If all the values are falsey, it will return the last value.

This returns the first truthy value or if all the values are falsey, it will return the last falsey value.
NOTE - Logical operators Use ANY data type, return ANY data type, follows short-circuiting.

EXAMPLES -

Nullish Coalescing operator ?? -

It is same as || operator, only diff is that, this operator short circuits based on Nullish values instead of falsey values.

NOTE - Nullish values are null and undefined only (not 0 or "").
